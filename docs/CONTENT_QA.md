# Content QA — Divine Timing

Guidelines for hero media, galleries, events, and shop so the site stays premium and consistent.

## Hero media

- **Image**: Recommended 1920×1080 or 16:9; high contrast so overlay text stays readable
- **Video**: Short loop, muted; avoid fast cuts. Provide a poster frame if possible
- **Uploadcare**: Use consistent filenames; avoid huge originals (compress before upload if needed)

## Galleries

- **Cover image**: Pick a strong first image; it represents the gallery in the grid
- **Order**: Curate order in admin; avoid duplicates
- **Captions**: Optional but helpful for accessibility and SEO

## Events

- **City / venue**: Use consistent format (e.g. "City · Venue Name")
- **Time zone**: Clarify in description or time field if events span regions
- **Thumbnail**: One clear image per event; aspect ratio consistent

## Shop

- **Product images**: Consistent background and crop across products
- **Descriptions**: Concise, premium tone; no keyword stuffing
