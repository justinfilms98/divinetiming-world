# Copy & Tone Guide

- **Tone**: Minimal, premium, confident. Human, not computer-generated.
- **Buttons**: Keep functional ("Save & Push Live", "Create Event"); refine only if a shorter label is clearer.
- **Empty states**: Calm, premium ("No upcoming events. Check back soon." / "No drops available yet.")
- **Form labels**: Concise, consistent (e.g. "Artist Name", "Ticket URL").
- **Errors**: Helpful, not technical ("Something went wrong. Please try again.").

### Do

- Use short sentences and clear CTAs
- Keep SEO titles/descriptions under 60 / 160 characters where it matters

### Don't

- Keyword stuffing
- Jargon in user-facing copy
- Overly casual or slang
