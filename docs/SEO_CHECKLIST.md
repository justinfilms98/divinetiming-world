# SEO Checklist

- **Metadata**: Each main route sets `title`, `description`, and Open Graph tags (use `page_settings` when present).
- **OG preview**: Test with [Open Graph debuggers](https://www.opengraph.xyz/) or similar; use a real deployment URL.
- **JSON-LD**: Validate output with Google’s Rich Results Test or Schema.org validator (paste the generated JSON-LD).
- **Canonical**: Ensure canonical URLs are set if you use multiple domains or params.

No tracking or analytics links are required by this checklist.
