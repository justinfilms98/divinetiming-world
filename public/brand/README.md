# Brand Assets

Place your brand assets here:

- `logo.svg` - Main logo
- `DT LOGO W.png` - White/light logo for header, branding, and OG when needed (use consistently; do not change design)
- `hero-eclipse.jpg` - Default hero background image (eclipse/cosmic theme)

These are referenced in the code as fallbacks. Hero logo is typically set per-page via Admin (Supabase storage); use `DT LOGO W.png` for static header/OG if desired.
